screen

screen basics

  • iniciar: ~ screen
  • listar sesiones: ~ screen -ls
  • attach: ~ screen -r [numero-sesion]

disponible en Linux / Mac / Cygwin

screen basics

  • Prefix: Ctrl-a
  • crea ventana: Ctrl-a c
  • lista ventanas: Ctrl-a "
  • dettach: Ctrl-a d

quick reference http://bit.ly/screen-quick-reference

tmux

tmux

  • igual que screen pero mejor.
  • cliente - servidor.

tmux basics

  • iniciar: ~ tmux
  • listar sesiones: ~ tmux list-sessions
  • listar clientes conectados: ~ tmux list-clients
  • attach: ~ tmux attach-session [-t numero-sesion]

disponible en Linux / Mac / Cygwin

tmux basics

  • Prefix: Ctrl-b
  • crea ventana: Ctrl-b c
  • dividir vertical: Ctrl-b "
  • dividir horizontal: Ctrl-b %
  • moverse de panel: Ctrl-b flecha
  • dettach: Ctrl-b d
  • ayuda: Ctrl-b ?
