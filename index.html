"Se me cortó la conexión mientras hacía la migración" - No más :)

screen

screen basics

  • iniciar: ~ screen
  • listar sesiones: ~ screen -ls
  • attach: ~ screen -r [numero-sesion]

disponible en Linux / Mac / Cygwin

screen basics

  • Prefix: Ctrl-a
  • crea ventana: Ctrl-a c
  • lista ventanas: Ctrl-a "
  • detach: Ctrl-a d

quick reference http://bit.ly/screen-quick-reference

Show time!

tmux

tmux

  • igual que screen pero mejor.
  • cliente - servidor.

tmux basics

  • iniciar: ~ tmux
  • listar sesiones: ~ tmux list-sessions
  • listar clientes conectados: ~ tmux list-clients
  • attach: ~ tmux attach-session [-t numero-sesion]

disponible en Linux / Mac / Cygwin

tmux basics

  • Prefix: Ctrl-b
  • crea ventana: Ctrl-b c
  • dividir vertical: Ctrl-b "
  • dividir horizontal: Ctrl-b %
  • moverse de panel: Ctrl-b flecha
  • detach: Ctrl-b d
  • ayuda: Ctrl-b ?

un poco de magia teamocil

crear sesiones automáticamente con tmux

¿qué es teamocil?

Herramienta para crear sesiones, ventanas y paneles en tmux usando YAML.

http://teamocil.com/

teamocil basics

				
~ gem install teamocil
~ mkdir ~/.teamocil
~ teamocil --edit sample
~ tmux
~ teamocil sample

disponible en Linux / Mac / Cygwin

~/.teamocil/cuatro.yml

				windows:
				- name: "cuatro-paneles"
				  root: "~/Code/sample/www"
				  layout: tiled
				  panes:
				    - cmd: "top"
				    - cmd: "tail -f /var/log/apache2/error.log"
				    - cmd: "tail -f /var/log/mail.log"
				    - cmd: "ping google.com"
